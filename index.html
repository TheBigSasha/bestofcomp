Bit Fields & Human-Like Techniques

Wherever a Collection would have been used, a custom Bitfield class is used instead. This stores the data in a single 32-bit integer (e.g. the Bitfield with the binary number 0...01101 contains 0, 2, and 3), allowing for quick access/modification using bitwise operations.
