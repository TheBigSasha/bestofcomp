Project: Motor Keyframer

A hardware and software system for motion control using Arduino and JavaFX. This system consists of control side software: MultiCOM a multi device serial terminal, Kinechrono a program for creating moves for the system, and a device side Arduino library that parses move messages to motor signals and runs its motors on a synchronized signal.
